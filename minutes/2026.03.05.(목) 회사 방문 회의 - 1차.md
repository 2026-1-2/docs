## 🗒️ 회의 내용
### 📎 안건 1: 프로젝트 방향성 확인
- **논의 내용**:
    - 제공된 기획서를 기반으로 진행하되, 종프2에서 작업한 사항이 바로 실 서비스에 적용되지는 않음.
    - 연말에 실제 적용 및 사용 예정 → 지금은 PoC/프로토타입 정도로 고려하여 작업하면 됨.
    - 일정 및 데드라인: 재료/서버 확정 후 개발 착수, 4~5월 개발 진행 예정

### 📎 안건 2: 프로젝트 진행 시 주의 사항
- **논의 내용**:
    - 스펙 원칙: 너무 특이한 스펙/희귀 장비는 지양 (차후 유지보수를 고려하여야 함.)
    - 카메라 구성 방향
      - 야외용을 우선하여 탐색하되, 불가피한 경우 실내용으로 대체. (실내용 활용하는 경우, 케이스 병용)
      - 카메라 회전이 가능해야 함.
      - 카메라 단가가 저렴한 경우(2~3만원), 성능 향상 대신 카메라 수를 늘려서 테스트. (실 운용 환경 고려)
    - 기타 주의사항
      - 영상 기록: 24시간 (상시) 기록 예정, 과거 n시간동안의 영상이 저장되어야 함. (n >= 24)
      - 시스템 접근에 대한 보안 처리 필요 (접근 권한 등)


### 📎 안건 3: 비용 집행 관련 사항
- **논의 내용**:
    - 재료는 수량/단가 파악 후 확정 예정
    - 최대한 유리한 방향으로 (카메라 대수 최대화 중심) 견적하여 본사로 전달.
    - 재료비 신청에서 "AI" 항목으로 LLM 사용 어려울 수 있음 -> 학교에 문의.
    
### 📎 안건 4: 실증을 위한 장소 선정
- **논의 내용**:
    - 드론 이착륙이 가능한 가용 비행장이 없음.
    - 드론/항공학과 등, 학내 유사 학과에 문의하여 장소 지원 가능 여부 확인
    - 장소 지원이 가능하면, 6월 발표 시 비행장에서 시연

### 📎 안건 5: 기타 사항
- **논의 내용**:
    - 드론 이착륙에 도움이 되는 정보를 함께 표시하면 좋음. (Optional)
      - 필수사항: 기상정보, 전파 맵 데이터
    - 저작권 위반하지 않는 범위에서, 유사 서비스 벤치마킹 가능.
